## Setup Instructions

### 1. Contact Form
To make the contact form work:
1. Go to [Formspree.io](https://formspree.io) and sign up.
2. Replace `https://formspree.io/f/xbjnqylg` in `index.html` with your URL.

### 2. WhatsApp Number
Current: `8090090051`. To change, Find & Replace in all HTML files.

### 3. Deploy to GitHub Pages
1. Upload these files to GitHub.
2. **Enable Pages** in Settings.
3. **Important**: For Voice features to work on mobile, the site must be served over **HTTPS** (which GitHub Pages does automatically).

## Customization
- **AI Responses**: Edit `voice-agent.js` to change what the AI says.
